# RepubliCaraquistApp
**RepubliCaraquistApp** es una aplicación web de análisis avanzado de la LVBP (Liga Venezolana de Béisbol Profesional), enfocada en los **Leones del Caracas**, desarrollada con Python, Streamlit y Supabase.   Integra estadísticas tradicionales y sabermétricas, automatización diaria y un módulo de inteligencia artificial para análisis contextual.
